아바타 이미지 정보
================

이 프로젝트는 avatars.jpg 파일이 필요합니다.

파일 사양:
- 파일명: avatars.jpg
- 위치: /frontend/public/avatars.jpg
- 형식: 5개의 아바타가 가로로 나열된 스프라이트 이미지
- 크기: 각 아바타 200x200px, 전체 이미지 1000x200px
- 사용법: CSS background-position으로 각 아바타 선택

아바타 ID별 위치:
- ID 1: 0% (첫 번째 아바타)
- ID 2: 25% (두 번째 아바타)
- ID 3: 50% (세 번째 아바타)
- ID 4: 75% (네 번째 아바타)
- ID 5: 100% (다섯 번째 아바타)

개발 시 임시 이미지나 실제 아바타 스프라이트 이미지를 추가해주세요.
